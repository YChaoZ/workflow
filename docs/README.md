# 工作流系统文档目录

本目录包含工作流系统开发过程中的所有文档，按照开发阶段和文档类型进行分类。

## 📁 目录结构

### 1. MVP开发文档 (`mvp/`)

按照MVP开发阶段组织的开发文档。

#### MVP1 - 基础功能
- 流程定义和部署
- 流程实例管理
- 任务管理
- BPMN设计器集成

#### MVP2 - 流程设计器 + 任务管理
- 流程设计器集成
- 任务管理完善
- 后端功能增强

#### MVP3 - 组织管理 + 表单设计
- 部门、用户、角色、权限管理
- 表单设计器
- 表单与流程集成

#### MVP4 - 高级功能
- 流程监控
- 流程统计
- 高级审批
- 系统配置

### 2. 测试文档 (`tests/`)

包含各阶段的测试计划、测试报告和测试结果。

### 3. 问题修复文档 (`fixes/`)

记录开发过程中遇到的问题和修复方案。

### 4. 环境搭建文档 (`setup/`)

包含项目环境搭建、配置和快速启动相关文档。

### 5. 优化文档 (`optimization/`)

技术优化、性能优化相关的方案和报告。
