【问题1：类型导入错误】

❌ 错误写法:
import axios, { AxiosInstance } from 'axios'

✅ 正确写法:
import axios from 'axios'
import type { AxiosInstance } from 'axios'

修复文件:
- src/api/request.ts
- src/store/modules/user.ts
- src/router/index.ts

【问题2：代理配置错误】

❌ 错误配置:
proxy: {
  '/api': {
    target: 'http://localhost:9099',
    rewrite: (path) => path.replace(/^\/api/, '')
  }
}

✅ 正确配置:
proxy: {
  '/api': {
    target: 'http://localhost:9099',
    changeOrigin: true
    // 不需要rewrite，后端路径包含/api
  }
}

修复文件:
- vite.config.ts
