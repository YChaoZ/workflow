## 🔧 常见问题

### 1. 启动失败
- 检查端口占用
- 检查数据库连接
- 检查Redis连接

### 2. 数据库迁移失败
- 检查Flyway配置
- 手动执行SQL脚本
- 重建数据库

### 3. 编译错误
- 检查Java版本（需要17+）
- 检查依赖版本
- 清理Maven缓存
